Methodology

Two independent checks were run, both filtering out the template repo (inno-se-toolkit/se-toolkit-lab-3, 1322 commits, 17 source files):

  1. File similarity — MD5 hashes of every source file, minus files identical to the template. Pairs with ≥50% identical non-template files are flagged.
  2. Git history — cross-student comparison of commit SHAs (critical), commit messages (high), and author emails (medium), excluding all 1322 template commits.

Raw flags were then manually investigated to distinguish real plagiarism from expected noise (lab-instructed commit messages, PR reviewer emails, trivially-constrained code).
