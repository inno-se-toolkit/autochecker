# Spec Folder

This folder contains product and delivery specs for the project upgrade.

It is intentionally separate from `specs/`, which already contains YAML lab definitions used by the autochecker runtime.

Files:
- `project-spec.md` — project-wide product and delivery spec
- `backend-ai-spec.md` — scope, ownership, and deliverables for the Backend + AI subteam
- `frontend-uiux-spec.md` — scope, ownership, and deliverables for the Frontend + UI/UX subteam

Subteam split:
- Backend + AI work together
- Frontend + UI/UX work together

Core framing:
- This is not a rewrite
- This is an upgrade of the existing autochecker into an AI tutor for students and an AI teacher-assistant for instructors
